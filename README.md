Projeto de portfólio desenvolvido em HTML, CSS e JavaScript para simular uma consultoria fitness automatizada, com foco em personalização, experiência do usuário e lógica de negócio.

FitConsult é um projeto interativo que demonstra como transformar um formulário em uma experiência inteligente de saúde e bem-estar, gerando um plano personalizado com treino, dieta e orientações com base no perfil do usuário.

O FitConsult transforma um formulário simples em uma experiência interativa que gera, com base no perfil do usuário, um plano estruturado com estimativa calórica, IMC, macros diários, cronograma de treino e protocolo alimentar com substituições.

O projeto foi criado para demonstrar, de forma prática, a criação de uma solução digital voltada para saúde e bem-estar, combinando UX, interface responsiva, lógica de personalização dinâmica e aplicação de regras específicas para diferentes perfis.

Funcionalidades principais: formulário com validação de dados, personalização por objetivo, perfil e nível de atividade, regras especiais para idosos e reabilitação, geração dinâmica de treino e dieta e uma interface voltada para apresentação e portfólio.

Tecnologias utilizadas: HTML5, CSS3 e JavaScript Vanilla.
